# Multiview Hand Pose CNN

Author: **Mattia Orlandi**

In this project, I implement and train the MultiViewHandPoseCNN model from [Ge et al.](https://arxiv.org/abs/1606.07253) (2016) on the [MSRA dataset](https://www.cv-foundation.org/openaccess/content_cvpr_2015/papers/Sun_Cascaded_Hand_Pose_2015_CVPR_paper.pdf) (2015).
